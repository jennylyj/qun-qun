我想要製作一個約時間的系統。

「群群」是一個有九個成員的自主學習群組，每個月都會有至少一次的固定聚會。在學期初相約能碰面的時間總是大工程。
我想要製作一個以 react vite 為主架構的網頁，之後部署到 github pages 以供大家使用。

1. 第一部分是姓名欄，我的組員可以輸入他的名字，接著後續的預約都會以他的名字為 state
所以會有 user_name 這個 state ，其中又有 user_code 這個屬性，紀錄名字的單字簡稱可供取用。

2. 第二部分是中間置中的「月檢視」行事曆，預設為本月（目前是 2025 01月）
且有左、右兩個箭頭，可以切換月份（並有橫移的動畫）
2-1. 行事曆的每個日期都是物件，他的顏色、和日期下方的數字，會隨著資料庫的內容改變。
2-2. 有兩種選取的模式：1. 是單日期選擇（點一下），2. 是用橫的選擇（拖曳） 3. 是按 shift
2-3. 選取之後，會跳出[O V X]（圈、三角形、叉）的對話筐來讓使用者選擇「該選取時間」自己是有空與否。其中圈的點值是+1 三角形是 0，叉是-1。這三個圖形請用合適的 icon 製作。
2-4. 使用者輸入完成後，系統將姓名、日期、圈叉三角形的選取資訊新增入資料庫中。
2-5. 重新整理畫面，日曆顯示新的資訊
